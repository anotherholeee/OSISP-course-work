+==============================================================================+
|          Схема взаимодействия с пользователем и внешней средой               |
|        «Корзина» для программ, использующих системный вызов unlink()         |
+==============================================================================+


+-------------------------------- Файловая среда ------------------------------+
|                                                                              |
|   ( $HOME )              ( Trash/files )           ( Trash/info )              |
|   рабочие файлы          удалённые объекты         *.trashinfo               |
|        ^                        ^                        ^                   |
|        |                        |                        |                   |
+--------|------------------------|------------------------|-------------------+
         |                        |                        |
         | Чтение при удалении    | Запись объекта         | Запись / чтение
         |                        |                        | метаданных
         |                        |                        |
+--------|------------------------|------------------------|-------------------+
|                    Системная программа (корзина)                             |
|                                                                              |
|         +----------------------+          +----------------------+           |
|         | libtrashhook.so      |          | trash-util           |           |
|         | перехват unlink()    |          | list, restore,       |           |
|         +----------+-----------+          | purge, info          |           |
|                    |                      +----------+-----------+           |
+--------------------|----------------------|-------------------------------+
                     |                      |
                     |                      |
+--------------------|----------------------|-------------------------------+
|                    |                      |                                  |
|  +-----------------v-----------+   +------v---------------------------+      |
|  | Прикладные программы        |   | libc          |  libdl             |      |
|  | (rm и др.)                  |   | unlink,       |  dlsym(            |      |
|  | LD_PRELOAD=libtrashhook.so  |   | rename, open  |  RTLD_NEXT)        |      |
|  +-------------+---------------+   +---------------+--------------------+      |
|                |                                                               |
|  +-------------v---------------+                                               |
|  | demo-cycle.sh               |                                               |
|  | setup-test-area.sh          |                                               |
|  +-----------------------------+                                               |
|                ^                                                               |
|                | вызов unlink / remove                                         |
+----------------|---------------------------------------------------------------+
                 |
+--------------------------------------------------------------------------------+
|                      Пользователь и среда запуска                              |
|                                                                                |
|    +-------------+         Ввод команды запуска         +-------------+        |
|    | Пользователь|------------------------------------->|  Терминал   |        |
|    |             |   LD_PRELOAD=... rm <файл>          |             |        |
|    |             |   ./trash-util list | restore       |             |        |
|    +------+------+                                     +------+------+        |
|           ^                                                   |                |
|           |         Результат работы и диагностика            |                |
|           +---------------------------------------------------+                |
|              (список корзины, «Восстановлено», ошибки)                         |
|                                                                                |
|           | Передача аргументов (путь, ID, команда)                             |
|           |                                                                    |
|           +------------------------+---------------------------+               |
|                                    |                           |               |
|                                    v                           v               |
|                          libtrashhook.so                 trash-util            |
+--------------------------------------------------------------------------------+


Основные потоки данных и команд:
  - пользователь вводит команду запуска в терминале;
  - прикладная программа (rm) с LD_PRELOAD вызывает unlink();
  - libtrashhook.so читает файл в каталоге $HOME пользователя;
  - объект переносится или копируется в ~/.local/share/Trash/files;
  - в каталог Trash/info записывается файл метаданных *.trashinfo;
  - для физического удаления и копирования используются libc и libdl;
  - пользователь запускает trash-util (list, restore, purge, info);
  - trash-util читает корзину и при restore возвращает файл в исходный путь;
  - диагностические сообщения выводятся пользователю через терминал.
